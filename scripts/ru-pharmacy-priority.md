# Приоритет торговых названий для РФ-аптеки

Справка для ревью ответов. Для каждой МНН — **топ-3 самых ходовых** в российской аптечной рознице 2025-2026. Если в ответе указан редкий ТН, заменить или дополнить на ходовой. Первое название в списке — самое массовое.

## Правила использования
1. Первое название в списке — основной массовый ТН. Второе и третье — альтернативы (можно перечислить через запятую).
2. Если в ответе указана редкая марка, заменить на основную.
3. Для ЛРС: сохранять сам растительный источник (алтей, подорожник и т.п.), но для торгового препарата использовать ходовой (Мукалтин, Гербион).
4. НЕ смешивать моно- и комбо-препараты (Амоксициллин ≠ Амоксиклав).
